✨ 功能特性
 * 📊 数据精准：基于系统级工具 vnStat 采集，支持昨日下载、上传及合计统计。
 * 📅 智能周期：支持自定义重置日（结算日），自动计算当前账单周期，完美处理大小月及闰年日期补齐。
 * 🌈 彩色进度条：根据流量使用百分比自动切换颜色（🟩 正常 / 🟧 预警 / 🟥 临界）。
 * 🕒 自动推送：内置 Cron 定时任务管理，每日凌晨 01:00 准时发送日报。
 * ⚙️ 极简配置：交互式安装界面，支持一键安装、配置覆盖、无损升级及彻底卸载。
 * 🐧 广泛兼容：支持 Debian/Ubuntu 及 CentOS/RHEL 等主流 Linux 发行版。

🚀 快速开始
1. 准备工作
在使用之前，您需要准备好：
 * 一个 Telegram Bot Token（通过 @BotFather 获取）。
 * 您的 Telegram Chat ID（通过 @userinfobot 获取）。
2. 一键安装 / 管理
执行以下命令进入交互式菜单：
curl -sSL https://raw.githubusercontent.com/您的用户名/仓库名/main/vnstat_tg.sh -o vnstat_tg.sh && chmod +x vnstat_tg.sh && ./vnstat_tg.sh

3. 菜单说明
 * 选项 1 (安装/重新配置)：全新安装依赖并设置参数。若已安装，会直接进入配置录入流程以覆盖旧设置。
 * 选项 2 (仅升级逻辑)：保留您的 Token 和 ID，仅更新脚本背后的计算逻辑和消息格式。
 * 选项 3 (卸载项目)：删除所有执行脚本、配置文件及定时任务，不留痕迹。
 * 选项 4 (手动测试)：立即发送一封当前状态的日报到您的 Telegram，用于验证配置是否正确。
📂 文件结构
 * /etc/vnstat_tg.conf：存储您的主机别名、Token、Chat ID、限额等私密配置。
 * /usr/local/bin/vnstat_tg_report.sh：实际负责数据采集与推送的执行脚本。
🛠️ 技术细节
 * 日期格式：强制执行 YYYY-MM-DD 格式补零对齐。
 * 单位换算：自动处理 MiB、GiB、TiB 之间的进制转换。
 * 依赖项：vnstat (核心数据), curl (API请求), bc (高精度计算), cron (调度)。
⚠️ 注意事项
 * 脚本会自动识别默认公网网卡，若您的环境特殊（如多网卡），请在安装时留意识别出的网卡名。
 * 初次安装后，vnStat 需要一点时间记录流量，建议运行 24 小时后再查看完整日报。
